Aplicacion que permite busacar recomendaciones de peliculas segun genero y libros segun autores.
Permite busacar peliculas por mas de un solo genero, pero permite buscar libros por un solo autor a la vez.
Se pueden buscar las peliculas y los libros al mismo tiempo, contando con un filtro para mostrar solo las peliculas o los libros.
Tanto la lista de autores de libros como los libros en si se obtienen consumiendo la API de Google Books.
Tanto los generos de peliculas como las peliculas en si se obtienen consumiendo la API de The Movie Database (TMDB).
Cuando se da click a las cartas de recomendacion redirecciona a la pagina de la pelicula en TMDB o la pagina del libro en Google Books.
